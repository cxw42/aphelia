APHELIA(1) General Commands Manual APHELIA(1)

NAME

aphelia —
the minimalist window manager

SYNOPSIS

Configuration and manipulation of software functionality should be done through editing the source code. No configuration files are provided.

DESCRIPTION

A light-weight, single-file window manager. Produces a small binary with low resource consumption. Useful for focused work and creating a distraction-free environment.

KEY-BINDINGS

The default keybindings:
Alt+a
Push window backwords
Alt+s
Pull window forward
Alt+q
Kill window
Alt+Return
Open (Suckless) Simple Terminal (st)
Alt+d
Open dmenu
Alt+Backspace
Kill window manager
Alt+Left ⟨click⟩
Move windows
Alt+Right ⟨click⟩
Resize windows

EXIT STATUS

The aphelia utility exits 0 on success, and >0 if an error occurs.
